# Chip

## Usage
```html
<mdc-chip label="chipName"
		leading="icon"
	    trailing="icon">
</mdc-chip>
```
## Parameters
| Option | Description | Required | Default |
|--|--|:--:|:--:|
| label		| The text behind the chip.					| X |				|
| leading			|	Material icon name for leading icon |		|				|
| trailing | Material icon name for trailing icon	| |				|
